# ⏱ Stopwatch Web App

This is a simple and responsive *Stopwatch* application built using HTML, CSS, and JavaScript.

### 🔹 Features:
- Start, Stop, and Reset the timer
- Glowing UI with animated background
- Mobile-friendly design

### 🔹 Technologies Used:
- HTML for structure
- CSS for styling (including glassmorphism and gradients)
- JavaScript for stopwatch logic

### 🔹 How to Use:
Just open index.html in your browser to use the stopwatch.
